ABOUT WoC

NITK Winter of Code is a X-week long online programme for students who are new to open source software development. The programme not only helps students to get involved in open source, but also prepares them for many open source summer programmes; Google Summer of Code being one of them.Come participate a enjoy working on Machine learning,Android ,Computer Vision,Gaming, Backend, Natural Language Processing ,Scrapping ,Cognition ,Front End, Deep Learning, Operating System ,DBMS, OOP,Networking, Compilers, Security ,Data Mining, Simulations ,Artificial Intelligence.

Why WoC?

Intro to Open Source

WoC provides a great opportunity to get acquainted with Github along with Git commands and contribute to open source efficiently.

Brush up your coding skills

If you love coding and want to learn about software development then WoC helps you to get a glimpse of it and gives you a head start.

Prepare for GSoC

With WoC, you get to know about how to select a project, interact with mentors and learn all other things that prepare you in the best way for the next GSoC.

Statistics: WoC 2018

WoC 2018 carried forward the legacy set by the 2017 edition; and in many ways went past the set milestones. It was staggering participation from students, many of them made their first contributions to open source during WoC, and mentors, who saw their projects surpass their own expectations by the end of the program.
